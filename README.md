# poker-dezhouapp-journal

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面的存储与分发用途。

## 项目简介

`poker-dezhouapp-journal` 主要收集和整理各类独立 HTML 页面，这些页面可能包含信息展示、工具界面、笔记存档等内容。所有页面均为静态 HTML 文件，不依赖后端服务，可直接通过浏览器访问。

## 目录说明

```
/
├── pages/        # 存放所有独立 HTML 页面
├── assets/       # 页面引用的静态资源（图片、样式、脚本等）
├── index.html    # 仓库首页入口（可选）
└── README.md     # 本文件
```

- `pages/`：每个子目录或文件代表一个独立页面，命名尽量反映内容主题。
- `assets/`：按需存放页面所需的 CSS、JavaScript、图片等资源文件。

## 页面归档说明

- 所有页面均为纯前端实现，无后端交互。
- 页面内容可能随时间更新，历史版本保留在 Git 提交记录中。
- 本仓库不保证页面内容的长期可用性或准确性，仅供存档参考。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进页面内容或修复问题。
- 新增页面请放置在 `pages/` 目录下，并尽量保持文件结构清晰。
- 请勿上传包含敏感信息、恶意代码或违反 GitHub 服务条款的内容。
- 本仓库由维护者不定期更新，不承诺固定的发布频率。
